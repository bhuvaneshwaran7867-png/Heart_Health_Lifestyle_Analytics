# 🫀 Heart Health & Lifestyle Analytics Dashboard

> **An interactive Streamlit dashboard for analyzing lifestyle and health factors associated with heart attack risk using data-driven insights.**

---

##  Overview

The **Heart Health & Lifestyle Analytics Dashboard** is an interactive Business Intelligence (BI) application built using **Python**, **Streamlit**, **Pandas**, and **Plotly**.

The dashboard helps healthcare organizations identify major heart attack risk factors by analyzing patient demographic, lifestyle, and clinical data. It enables stakeholders to explore trends, compare risk factors, and make informed preventive healthcare decisions through interactive visualizations.

---

##  Problem Statement

Heart attacks are increasingly being reported across different age groups due to changing lifestyles and health conditions.

Healthcare organizations need an analytical solution to answer questions such as:

* Which age group is at the highest risk?
* Does stress, sleep, or lack of exercise increase heart attack risk?
* How do BMI, cholesterol, smoking, alcohol consumption, and fast-food habits affect heart health?
* Which cities or occupations have a higher number of high-risk patients?
* What preventive measures can reduce the risk of heart attacks?

This dashboard addresses these questions through interactive visualizations and business insights.

##  Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/Heart-Health-Lifestyle-Analytics.git
```

### 2. Navigate to the Project Folder

```bash
cd Heart-Health-Lifestyle-Analytics
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
streamlit run app.py
```

The application will open in your default web browser.
